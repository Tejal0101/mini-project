# mini-project
 Member 1: Emotion Detection & Backend Integration
  Responsibilities:- Implement face and emotion recognition using Face-api.js or TensorFlow.js.- Capture webcam input and detect mood.- Set up backend using Node.js + Express.- Create APIs for mood detection, song    retrieval, and playlist saving.- Design MongoDB/Firebase database schema.
  Deliverables: Emotion detection module, backend APIs, database setup.
 Member 2: Music Player & Playlist Management
  Responsibilities:- Develop music player using HTML Audio API or Spotify/YouTube embed.- Add play/pause, next, and volume control features.- Enable users to create, edit, and save playlists.- Categorize songs by     emotion and genre.
  Deliverables: Functional music player, playlist management, genre-song mapping.
 Member 3: Frontend UI/UX & Integration
  Responsibilities:- Design frontend using HTML, CSS, and JavaScript (or React).- Create pages: Home, Player, Playlist Editor.- Integrate backend APIs and display detected emotions dynamically.- Implement             responsive design and animations.
  Deliverables: User interface, API integration, complete web app layout
